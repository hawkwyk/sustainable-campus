
## 页面功能详述

### index.html - 主页
- **导航栏**: 四个主要栏目导航
- **Hero区域**: 课程介绍 + p5.js粒子动画背景
- **特色项目展示**: 轮播展示6个重点改造项目
- **数据统计**: 项目成果数据可视化（ECharts.js）
- **课程亮点**: 图文结合的介绍卡片
- **页脚**: 版权信息和联系方式

### projects.html - 项目故事页面
- **项目分类筛选器**: 6个项目类型的标签切换
- **项目网格展示**: 每个项目的卡片式布局
- **项目详情模态框**: 点击卡片弹出的详细信息
- **图片轮播**: Splide.js实现的项目图片展示
- **微信公众号链接**: 跳转到相关推送文章

### tour.html - 交互式地图页面
- **地图容器**: 高德地图显示北京大学校园
- **标记分类系统**: 不同项目类型使用不同图标
- **左侧控制面板**: 
  - 新增标记按钮
  - 标记类型筛选
  - 已有点位列表
- **标记信息窗口**: 点击标记显示的项目详情
- **表单模态框**: 新增标记的信息输入表单
- **GitHub API集成**: 自动提交标记数据到仓库

### about.html - 关于我们页面
- **项目历史**: 时间轴形式展示项目发展历程
- **团队成员**: 卡片式展示团队成员信息
- **联系方式**: 联系表单和社交媒体链接
- **公众号二维码**: 微信扫码关注

## 核心功能模块

### main.js - 主要JavaScript功能
- 页面导航和路由管理
- 动画效果控制（Anime.js）
- 模态框和表单处理
- 地图初始化和标记管理
- GitHub API交互逻辑
- 响应式布局适配

### config.js - 配置文件
- 高德地图API密钥配置
- GitHub仓库信息（用户名、仓库名）
- 项目分类和图标配置
- 页面内容文本配置
- 图片资源路径配置
- 团队成员信息配置

### GitHub集成
- 使用GitHub API创建Issue或提交PR
- 自动处理用户上传的标记数据
- 图片文件上传到repository的contents文件夹
- 配置GitHub Secrets存储敏感信息

## 技术栈
- **前端框架**: 原生HTML/CSS/JavaScript
- **样式框架**: Tailwind CSS
- **动画库**: Anime.js, p5.js
- **数据可视化**: ECharts.js
- **轮播组件**: Splide.js
- **地图服务**: 高德地图API
- **部署平台**: GitHub Pages
- **版本控制**: Git/GitHub
