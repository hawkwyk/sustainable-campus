# 可持续校园实践网站交互设计

## 核心交互功能

### 1. 主页交互
- 特色项目轮播展示：自动播放，用户可手动切换
- 数据统计动画：数字递增效果展示项目成果
- 项目卡片悬停效果：3D倾斜和阴影变化

### 2. 项目故事页面交互
- 项目分类筛选器：点击分类标签筛选对应项目
- 项目卡片展开：点击项目卡片弹出详情模态框
- 图片轮播：每个项目包含多张照片轮播
- 微信公众号链接：点击跳转到对应推送文章

### 3. Tour地图页面交互
- 高德地图集成：显示北京大学校园地图
- 分类标记系统：不同项目类型使用不同图标
- 标记点点击：弹出信息窗口显示项目详情和图片
- 新增标记功能：
  - 点击"新增标记"按钮激活标记模式
  - 点击地图位置获取经纬度
  - 弹出表单：标题、描述、分类、图片上传
  - 通过GitHub API自动提交到仓库contents文件夹

### 4. 关于我们页面交互
- 团队成员卡片：悬停显示详细信息
- 公众号二维码：点击放大显示
- 联系表单：验证和提交反馈

## 用户体验流程

### 项目浏览流程
1. 用户进入主页了解课程概况
2. 点击"项目故事"查看具体项目
3. 使用分类筛选找到感兴趣的项目
4. 点击项目卡片查看详情和图片
5. 通过链接访问微信公众号了解更多

### 地图标记流程
1. 进入Tour页面查看已有标记
2. 点击"新增标记"按钮
3. 在地图上选择位置
4. 填写表单信息并上传图片
5. 提交后自动同步到GitHub仓库

## 响应式设计
- 移动端优化的触摸交互
- 平板端适配的布局调整
- 桌面端的鼠标悬停效果
